# RookieHosting
A project to teach people the basics to hosting things on the internet, and generally make it easier at points.


## Available Documentation
The goal of this project is to have quite literally as many different server technologies as possible available to the public to utilize on their own projects. Of course, we must start somewhere. Below is a list of server technologies that we have actively maintained documentation/tools for:
- Beam.MP
- Port Forwarding
- Hopefully more soon!

### Possible to be documented
Me (and hopefully others), myself and I aren't done here; I would like to do lots with this project. As I have already stated once, we must start somewhere. Here is a list of different server technologies which are next on the to-do list:
- Minecraft Servers
- Web Hosting
- IP Addresses
- Logs/Records
- Server Commands
- Core Functions of a Server
- and more to be decided!
